MeMailSettings

An application that helps to enable some features for Nokia N9 MfE and/or Email accounts.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to MeMailSettings page.

MeMailSettings can make your Nokia N9 Email client even better with folder's mapping and few more additional settings. Device reboot might be needed to enable some new features. If you need any help please write an email to me.
